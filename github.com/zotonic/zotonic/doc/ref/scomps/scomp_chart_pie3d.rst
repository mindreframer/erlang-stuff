Show a pie chart with 3D effect.

This scomp is just a convenient interface to the :ref:`{% chart_pie %}
<scomp-chart_pie>` scomp with the ``threed`` argument set to true.

For example::

  {% chart_pie3d data=[["firefox",23],
                       ["internet explorer", 67],
                       ["safari",4],
                       ["chrome",3],
                       ["other", 3]]
  %}

Gives::

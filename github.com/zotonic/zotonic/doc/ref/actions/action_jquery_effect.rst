
.. include:: meta-jquery_effect.rst

Trigger various jQuery effects on the target element. Mostly, each of these effects have their own action as a shortcut, for example :ref:`action-show`, :ref:`action-hide`.

Arguments:

* type - one of `show`, `hide`, `remove`, `slide_toggle`, `toggle`, `set_class`, `add_class`, `remove_class`, `fade_in`, `fade_out`, `slide_up`, `slide_fade_out`, `slide_fade_in`, `disable`, `enable`, `effect`, `animate`.

* speed
* class
* easing      
* effect
* options
